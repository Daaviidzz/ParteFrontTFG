Requisitos del sistema

💻 Windows 10 / macOS 12 / Linux
🧠 4 GB RAM
💾 500 MB espacio libre
🖥️ OpenGL 4.3 o DirectX 11
RECOMENDADOS
💻 Windows 11
🧠 8 GB RAM
💾 1 GB espacio libre
🌐 Conexión a internet (API)

Estado de descarga

1
Proyecto Unity en Plastic SCM

El juego de Unity no está publicado en GitHub. El equipo lo gestiona en Plastic SCM durante el desarrollo.

2
Build pendiente de publicar

Cuando haya una versión exportada del juego, se añadirá aquí el enlace de descarga del ZIP.

3
Extraer el ZIP

Cuando el ZIP esté disponible, habrá que extraerlo en una carpeta y mantener la estructura interna de archivos.

4
Conexión con la API

La versión final del juego se conectará a la API oficial para login, partidas, catálogo y datos de jugador.

5
Ejecutar el juego

En Windows, la build tendrá un ejecutable del juego cuando esté publicada. En tu primera partida podrás elegir personaje y Ibermon inicial.

✅ Mientras tanto, puedes explorar el catálogo y probar la API desde esta web.

Preguntas frecuentes

¿El juego es gratuito? +

Sí, Ibermon es un proyecto de TFG completamente gratuito y de código abierto.

¿Funciona sin internet? +

El juego puede funcionar sin internet en modo básico, pero el guardado en la nube y la sincronización del equipo requieren conexión a la API.

¿Puedo contribuir al proyecto? +

El desarrollo del juego se gestiona en Plastic SCM por el equipo. La web y la API sí tienen repositorios separados.
