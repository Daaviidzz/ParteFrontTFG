Requisitos del sistema

MÍNIMOS
💻 Windows 10 / macOS 12 / Linux
🧠 4 GB RAM
💾 500 MB espacio libre
🖥️ OpenGL 4.3 o DirectX 11
RECOMENDADOS
💻 Windows 11
🧠 8 GB RAM
💾 1 GB espacio libre
🌐 Conexión a internet (API)

Pasos de instalación

1
Ir al repositorio de GitHub

Accede al repositorio oficial del proyecto en GitHub donde encontrarás el código fuente y los releases compilados del juego.

🔗 Abrir GitHub →
2
Descargar el release

En la sección Releases del repositorio, descarga la versión más reciente del juego para tu sistema operativo. El archivo vendrá comprimido en .zip.

3
Extraer el archivo

Extrae el contenido del ZIP en una carpeta de tu elección. No muevas los archivos internos, deben permanecer en la misma estructura de carpetas.

4
Configurar la API (opcional)

Para el guardado en la nube necesitas tener la API corriendo. Puedes usar Docker:

docker-compose up -d

O conectarte al servidor oficial si está disponible. Sin API, el juego funciona solo en modo local.

5
¡Ejecutar el juego!

Abre el ejecutable Ibermon.exe (Windows) o el equivalente en tu sistema. En tu primera partida podrás elegir personaje y Ibermon inicial.

Preguntas frecuentes

¿El juego es gratuito? +

Sí, Ibermon es un proyecto de TFG completamente gratuito y de código abierto.

¿Funciona sin internet? +

El juego puede funcionar sin internet en modo básico, pero el guardado en la nube y la sincronización del equipo requieren conexión a la API.

¿Puedo contribuir al proyecto? +

¡Claro! El código está en GitHub. Puedes abrir issues, proponer features o hacer pull requests.
